Properties Menu

The properties menu allows the user to view and modify properties of the selected component.

All built-in circuit components in Logisim come with preset properties, such as the direction they initially face and how many inputs/outputs they have. The properties menu is where you can view and edit these properties so that the component fits your circuit design. Some properties include:

  • The direction the component faces

  • The size of the component

  • How many inputs a component takes

  • The name of the component

To view a component's properties, click the component while in select mode. The component's properties will automatically appear in the properties menu.
Note: If you are looking to change the look or functionality of a component, look in the properties menu first.
