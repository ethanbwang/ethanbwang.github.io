Building a Synchronous Counter

Learn how to build a basic synchronous counter in Logisim.

Counters are one of the most basic sequential circuits used to introduce the clock and other sequential ideas. This short guide walks you through building a simple four-bit synchronous counter that counts once per clock cycle.
  1. In the top bar, click the register .
  2. Place the register in circuit view.
  3. In the Properties menu:
    1. Set Data Bits to 4.
    2. Set Label to R0.
    3. Set Show in Registers Tab to Yes.
  4. In the Design menu, click Wiring > Clock.
  5. Place the clock to the left of the clock input of the register (triangle).
  6. Wire the clock to the clock input of the register.


  7. In the Design menu, click Arithmetic > Adder.
  8. Place the adder to the left of the register's D input.
  9. In the Properties menu:
    1. Set Data Bits to 4.
  10. Wire the adder output to the register's D input.
  11. Wire the register's Q output to the adder's first input.


  12. In the Design menu, click Wiring > Constant.
  13. Place the constant to the left of the adder's second input.
  14. In the Properties menu:
    1. Set Data Bits to 4.
    2. Set Value to 0x1 (1 in hex).
  15. Wire the constant to the adder's second input.


Congratulations! You have successfully built a working four-bit synchronous counter.
